# Marketing Attribution & ROI Analytics Platform

## Business Problem
Marketing teams often optimize for clicks instead of revenue, leading to
inefficient budget allocation.

## Solution
Built an attribution and ROI analytics framework that connects marketing
touchpoints to conversions and revenue.

## Tech Stack
Python, Marketing Analytics concepts, GA-style data, CRM-style data

## Key Insights
- Identified high-ROI channels
- Demonstrated limitations of last-touch attribution
- Enabled data-driven budget recommendations

## Future Enhancements
- Multi-touch path-level attribution
- A/B testing integration
- Real-time dashboards
